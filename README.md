# AI Roadmap Generator 

An AI-powered web application that generates personalized learning roadmaps based on your goals, current skills, and available time.

---

##  Features

-  User Authentication (Signup/Login with JWT)
-  AI-generated personalized roadmaps
-  Phase-wise structured learning plan
-  Skills, Projects, and Resources included
-  Save roadmaps to database
-  View detailed roadmap page
-  Delete roadmaps
-  Copy roadmap to clipboard
-  Clean and responsive UI

---

##  Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Other
- JWT Authentication
- OpenRouter API (LLM for roadmap generation)
- React Hot Toast

---

##  How it Works

1. User signs up / logs in  
2. Inputs:
   - Target Role  
   - Current Skills  
   - Time Availability  
3. AI generates a structured roadmap  
4. Roadmap is saved and displayed  
5. User can view, copy, or delete it
